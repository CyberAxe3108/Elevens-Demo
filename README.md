## Overview
This project is a demo version of the **Elevens Card Game**, currently under development. Elevens is a single-player card game where the goal is to remove cards from the table by finding valid combinations that add up to eleven, or specific face-card combinations.

This demo focuses on building the core game logic and mechanics using object-oriented programming concepts.

## Current Features
- Standard 52-card deck
- Shuffle functionality
- Deal cards to the table (up to 9 cards)
- Basic card and deck classes
- Game structure setup

## Rules (Simplified)
- Number cards can be removed if two cards add up to **11**
- Face cards follow special rules (to be implemented)
- The game ends when no more valid moves are possible
